## 🛠️ What I Built

### Hardware (Projects 1–5) — HDL
- All logic gates built from a single NAND gate
- A fully functional 32-bit ALU
- Memory hierarchy: Registers → RAM8 → RAM64 → RAM16K
- A complete CPU and integrated Memory system

### Software (Projects 6–12) — Java & Jack
- **Hack Assembler**: translates Assembly to binary machine code
- **VM Translator**: two-stage virtual machine translator
- **Jack Compiler**: full two-stage compiler (Tokenizer + Code Generator)
- **Jack OS**: complete standard library (Math, Screen, Keyboard, String, Array, Sys...)
